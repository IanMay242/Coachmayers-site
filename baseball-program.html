Questions

Common Questions.

What equipment do I need? +
You need access to a performance gym with: barbell and plates, trap bar or hex bar, dumbbells, medicine balls (8–12 lb), resistance bands, a sled, a box jump platform, and a 60-yard sprint surface. Substitutions are provided for most equipment if something isn't available.
Is this only for position players or can pitchers use it too? +
This program is built specifically for position players — the speed, power, and rotational demands of hitters and fielders. Pitchers have different development priorities. A pitcher-specific Mayers Method program is in development.
What level is this program designed for? +
High-level high school, college, and professional players. Athletes should have some base strength training experience and be comfortable with barbell movements before starting Phase 1. A lower-equipment version for developing athletes is coming soon.
How many days per week and how long are sessions? +
Three days per week. Recommended: Monday, Wednesday, Friday — or any three non-consecutive days. Each session runs approximately 75 to 90 minutes depending on the phase.
Can I run this during the season? +
No — this is an off-season performance development program, not an in-season maintenance program. Ideally start 16+ weeks before your season begins so you peak in Phase 4 heading into opening day.
How does delivery work? +
The program is delivered instantly via Bridge Athletic — a professional coaching app used by elite programs and coaches worldwide. After purchasing you'll receive access instructions and the Arm Care Guide PDF. Download the Bridge Athletic app, log in, and your full program is ready to go.
How is this different from a random workout I find online? +
This is a fully periodized 16-week system built on the same methodology used with professional athletes — not a workout PDF with exercises thrown together. Every phase has a physiological purpose. Every session builds on the last. Every exercise is selected for its transfer to baseball performance.
